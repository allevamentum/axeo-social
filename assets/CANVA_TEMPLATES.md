# AXEO — Canva Template Specifications

## Brand Tokens (Use in Every Design)

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Surface (bg) | #08090C | All backgrounds |
| Elevated | #12141A | Card backgrounds |
| Card | #1C1F28 | Content blocks |
| Border | #2A2E3A | Dividers, outlines |
| Cyan | #00FFD5 | Primary accent, CTAs, highlights |
| Cyan Muted | #009B8A | Secondary accent |
| Retinal | #FF4500 | Alerts, key stats, emphasis |
| Gold | #E09B3D | Warnings, pricing |
| Data Blue | #3B82F6 | Charts, data points |
| Success | #22C55E | Positive results |
| Text Primary | #EAECF0 | Headlines |
| Text Secondary | #8A90A4 | Body text |
| Text Muted | #4A5068 | Labels, captions |

### Fonts
| Usage | Font | Weight | Tracking |
|-------|------|--------|----------|
| Headlines | Inter | 300 (Light) | +0.04em |
| Body | DM Sans | 300 (Light) | default |
| Overlines/Tags | IBM Plex Mono | 500 | +0.18em, UPPERCASE |
| Data/Numbers | IBM Plex Mono | 400 | default |

### Logo
- Use the X-mechanism icon (cyan arms + retinal dot)
- Small: bottom-right corner, 40x40px
- Watermark: 50% opacity

## Quick Setup in Canva

1. Create a "Brand Kit" in Canva with the colors and fonts above
2. Create each template as a reusable Canva template
3. For each week's content:
   - Duplicate template
   - Swap text from the weekly content JSON
   - Export as PNG (static) or MP4 (animated)
4. Save to `/assets/` folder organized by week

## Export Settings
- Instagram/TikTok: PNG, 1080x1920 or 1080x1080
- X media cards: PNG, 1200x675
- YouTube thumbnails: JPG, 1280x720
- File naming: `W01-MON-01-tiktok.png`
